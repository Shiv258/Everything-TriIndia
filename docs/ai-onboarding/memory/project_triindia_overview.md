**TRIINDIA Hospitality** is a Delhi partnership firm with 5 budget/mid-range hotels currently running on pen and paper. Shiv is building a self-hosted digital operating system covering bookings, front-desk, guest data, billing, marketing, and guest communication. Domain: triindia.in (also triindiahospitality.com is in active use).

**The 5 hotels:** J Residency (pilot — build everything here first), Hotel Satwah 29, Hotel Ashram View, Hotel Preet Place, Hotel Samrat Residency. Hotels 2–5 are Phase 1.5 (after pilot proves out).

**Three partners (unanimous decisions):** Mr. Hansraj Lakhmani, Mr. Habib Mohammad, Mr. Mohd Aftab. Communication via WhatsApp group with all three + Shiv.

**Phase 1 timeline:** 50 days starting Day 0 = 10 May 2026 (per KB title) — Day 25 demo ~28 May, Day 50 handover ~22 June. Tier 2 (Growth) scope + AI Upsell Agent + Historical Register OCR Migration add-ons. Google Reviews automation explicitly declined.

**Why:** Reduce OTA dependence (currently 20–30% commission), digitize paper registers, run a marketing engine for the March–July summer dip, preserve the Kalakar tradition (auto/cab driver referrals) but make Diwali reward selection data-driven instead of memory-based.
